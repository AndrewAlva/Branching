Process to create, merge and delete branches

  1. git branch [new-branch-name]
  2. git checkout [just-created-branch-name]
  3. Make the changes you need:
    • git add --all
    • git commit -m "Put your commit here"
    • git push origin [branch-name]
  4. git checkout master
  5. git merge [branch-to-merge-name]
  6. git add --all
  7. git commit -m "Commit you just merged the branch"
  8. git push origin master
  9. git branch -D [branch-name-to-delete]
